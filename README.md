# schedule-app

個人開発のスケジュール管理アプリです。タスク管理・時間割・教習（自動車学校）のスケジュールを一つのアプリでまとめて確認ができます。

## 機能一覧

### タスク
- タスクの追加・完了切り替え・削除
- 期限（日付・時刻）の設定と、期限までの残り日数表示（「今日まで」「明日まで」「期限切れ」など）
- タスクごとに6色（赤・橙・黄・緑・青・紫）のラベルカラーを設定可能
- 期限が近いタスクが上に来るよう自動で並び替え

### 時間割
- 月〜金・1〜6限のグリッドで授業を登録
- 授業名・教室・カラーラベルを設定
- 今日の曜日をハイライト表示

### 教習
- 自動車学校の「学科」「技能」を登録・削除
- 予定日時とメモ（例：第1段階 みきわめ）を記録
- 予定/過去で自動的にセクション分け、当日・翌日・何日後かのバッジ表示

### 設定
- ダークモード切り替え
- 文字サイズ（小・中・大）の変更

### データ同期
- Firebase Firestore を使い、入力したデータをリアルタイムに保存・同期
